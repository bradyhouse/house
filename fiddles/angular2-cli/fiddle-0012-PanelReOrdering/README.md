### Description

Angular cli fiddle exploring how to create a vertical collection of windows (or panels) that can be
re-ordered by moving (dragging and dropping) the window's title bar.  It combines:
 
  * angular (v4.0.0)
  * bootstrap (v3.3.7)
  * jquery (v3.2.1)
  * select2 (v4.0.3)
  * lodash (v4.17.4)
  * bootswatch (v3.3.7)
  * ngrx/core (v1.2.0)
  * ngrx/store (v4.0.3)
  * ngrx/effects (v4.0.5)
